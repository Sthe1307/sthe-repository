O que é Python?
Python é uma linguagem de programação versátil, criada por Guido van Rossum em 1991. Conhecida por sua sintaxe simples, ideal para iniciantes. Python é usada em várias áreas, como:
  • Web (Django, Flask)
  • Ciência de Dados (Pandas, NumPy)
  • Inteligência Artificial (TensorFlow, Scikit-learn)
  • Automação e Scripts
É uma das linguagens mais populares do mundo, com uma vasta comunidade e bibliotecas poderosas.
Variáveis
Variáveis em Python armazenam dados e não precisam de tipo explícito. Exemplos:
  • idade = 25 (Número inteiro)
  • nome = "João" (Texto ou string)
  • pi = 3.14 (Número decimal ou float)
Laços de Repetição
Python suporta laços como for e while para repetir blocos de código:
  • for i in range(5): print(i) (Imprime números de 0 a 4)
  • while x < 10: x += 1 (Incrementa enquanto x for menor que 10)
Funções
Funções permitem organizar e reutilizar código. Defina com def:
  • def soma(a, b): return a + b (Soma dois números)
  • def saudacao(nome): print(f"Olá, {nome}!") (Exibe uma mensagem personalizada)
